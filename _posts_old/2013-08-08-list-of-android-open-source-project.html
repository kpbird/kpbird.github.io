After browsing above long list a contributors, you might be thinking "How to get list of contributors ?"

Steps to fetch contributors 

Step 1 : Download Android Source code. (Download instruction)
Step 2 : Open Terminal / Console
Step 3 : Go to folder <Android Source>/frameworks/base/core
Step 4 : Execute following command

git shortlog -s -n

If you want to name with email id then use following command

git shortlog -s -n -e

References
1. http://source.android.com/source/initializing.html
2. https://www.kernel.org/pub/software/scm/git/docs/git-shortlog.html
3. http://source.android.com/source/contributing.html
